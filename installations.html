Installation

Installation Process

The following process will be used to install admin dashboard using package manager.

  1. Install Node.js : Please install latest version of Node.js from https://nodejs.org
  2. Install npm from https://npm.com
  3. Extract downloaded project files : Extract the admin dashboard to your suitable directory or folder.
  4. Access console : Open node.js console.
  5. In console, navigate to the location where main folder is extracted.
  6. Navigate to the Source folder inside main folder.
  7. Run npm install command to install package dependencies.
  8. Run npm run watch to start compiling assets or npm run dev for production use.
  9. The template is ready to use!
