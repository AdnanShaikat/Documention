Style Customization

SASS Structure

Midone use SASS as a default css preprocessing, you can found all the SASS files in one single place at /sass/* directory. Every single component SASS file are imported in /sass/app.scss. You can modify this file to add or remove components depends to your needs. Here is the part things you need to know about the /sass/app.scss structure and functions.

  1. Fonts: This section is supposed to import all the template used fonts.
  2. Breakpoint: This file is contains screen breakpoint variables that used by the template components.
  3. 3rd Parties: This section supposed to import all 3rd parties SASS or CSS files.
  4. Animations: This section supposed to import all custom animations used by the template.
  5. Components: This section supposed to import all component SASS files used by the template.
