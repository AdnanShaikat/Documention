File Structure

File Structure

After downloading the template, you will find 3 files in it. If you want to directly use this template without making modifications you can immediately use the file in the compiled directory. This directory contains compiled and minified file that ready for production. But if you want to do some modifications like adding or removing new components or parts you will be better to use the files from source directory.

The file tree below will describes more clearly about the file structure.

Main
Assets
Dist
node_modules
Pages
Src

Package.json
