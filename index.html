Introduction

Midone gives you everything you need to create your Admin Dashboard. There are a few things like colors, components, and layouts that carefully chosen and designed for every single page. The template are made with bootstrap-5.0 file which has many usefull utility classes and great SASS support.

The download files will contains 3 directories, the project source files and documentation to help you starting to use this template. It is created to save your time so you can take care the other parts. Every components are completely modular and written with SASS. Thank you for purchasing this template.
